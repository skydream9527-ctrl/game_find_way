# 游戏设计文档 - Find Way

## 概述

《Find Way》是一款横版平台跳跃游戏，玩家通过点击/按住屏幕控制角色跳跃，在奇幻冒险世界中穿越 10 个章节共 100 个关卡。

## 技术架构

| 项目 | 选择 |
|------|------|
| 技术栈 | KMP + Android 原生 Canvas |
| 架构模式 | Android Clean Architecture |
| UI 框架 | Compose Multiplatform (Android 端) |
| 依赖注入 | Koin |
| 数据存储 | DataStore (配置/进度) |
| 音频 | Android MediaPlayer + SoundPool |

### 模块结构

```
project/
├── app/                  # Android 入口，DI 配置
├── core/                 # 共享工具、错误类型、扩展函数
├── domain/               # 纯 Kotlin：游戏引擎、用例、仓储接口
├── data/                 # 仓储实现、本地存储
└── presentation/         # UI 层：Compose 界面、Canvas 渲染、ViewModel
```

### 依赖规则

```
app → presentation, data, domain, core
presentation → domain, core
data → domain, core
domain → core
core → 无依赖
```

## 游戏流程

```
启动页 → 主菜单 → [开始游戏 | 角色属性 | 设置]
  → 章节选择 → 关卡选择 → 关卡加载 → 游戏中
    → 成功 → 关卡完成 (选择下一关/返回章节)
    → 失败 → 重新开始
```

## 核心机制

### 移动系统
- 关卡加载完成后 1-2 秒缓慢加速到最大速度
- 达到最大速度后自动匀速前进
- 玩家只需操作跳跃

### 跳跃系统
- **点击**：触发基础跳跃
- **按住蓄力**：按住时间决定跳跃高度/距离（0.5-2 秒）
- **二段跳**：解锁后在空中再次点击触发
- **超长跳跃**：道具效果，跳跃距离 x2 或 x3

### 碰撞检测
- AABB 矩形碰撞检测
- 脚部落在平台上 → 站立
- 侧面撞到平台边缘 → 失败
- 掉落屏幕底部 → 失败

## 平台生成

| 属性 | 简单关卡 | 困难关卡 |
|------|----------|----------|
| 平台数量 | 20-30 个 | 40-50 个 |
| 平台间距 | 小 | 大 |
| 平台宽度 | 宽 | 窄 |
| 高度变化 | 平缓 | 剧烈 |
| 移动平台 | 无 | 后期出现 |

## 道具系统

### 即时消耗型
| 道具 | 效果 | 持续时间 |
|------|------|----------|
| 🍃 羽毛 | 跳跃高度 +50% | 10 秒 |
| 🦋 蝴蝶 | 二段跳 | 15 秒 |
| ⚡ 闪电 | 移动速度 +30% | 8 秒 |
| 🛡️ 护盾 | 免疫一次失败 | 单次 |

### 永久解锁型
| 道具 | 效果 |
|------|------|
| 💎 宝石 | 解锁二段跳技能 |
| 🌟 星星 | 基础跳跃力 +10% |

### 生成规则
- 每关卡随机 3-8 个道具，10-30 个金币
- 道具生成在平台上方，需跳跃获取
- 永久道具仅在章节最终关卡出现
- 金币可累积用于解锁外观

## UI 设计

### 启动页
- 游戏 Logo + 标题
- 开始游戏按钮
- 角色属性入口
- 设置入口

### 角色属性页
- 角色头像 + 名称 + 等级 + 金币数量
- 基础属性条：生命值、跳跃力、移动速度
- 已解锁技能列表
- 外观/皮肤选项

### 游戏页面
- 全屏 Canvas 渲染游戏画面
- 顶部 HUD：金币数、当前章节/关卡
- 底部提示区：蓄力指示器（按住时显示）

### 章节选择页
- 章节卡片网格/列表
- 已解锁/进行中/未解锁状态
- 每章显示进度（X/10 关完成）

## 角色设计

矢量卡通风格：圆润身体 + 大眼睛 + 鲜艳配色
- 站立/奔跑：正常姿态
- 跳跃：手臂张开、嘴巴兴奋
- 蓄力：身体微蹲、表情专注

## 数据存储

### 本地保存
- 游戏进度（章节/关卡解锁状态）
- 角色属性（生命值、跳跃力、移动速度等级）
- 金币数量
- 已解锁技能
- 设置（音量、震动等）

### 数据格式
```kotlin
data class GameProgress(
    val currentChapter: Int,
    val unlockedLevels: Map<Int, List<Int>>,
    val completedLevels: Map<Int, List<Int>>,
    val totalCoins: Int
)

data class CharacterStats(
    val health: Int,
    val jumpPower: Float,
    val moveSpeed: Float,
    val unlockedSkills: Set<SkillType>
)
```

## 音频系统

- 每个章节独立背景音乐 (BGM)
- 动作音效：跳跃、落地、拾取金币、拾取道具
- 状态音效：关卡完成、游戏失败、按钮点击
- 支持音量调节和静音
